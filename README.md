# 🌧️ Rain Prediction Model using Machine Learning

A simple and effective Machine Learning project that predicts whether it will rain or not based on weather conditions.

---

## 📌 Project Overview

This project uses historical weather data to train a Machine Learning model that can predict rainfall. It helps in understanding how different weather features (like humidity, temperature, wind speed, etc.) influence rain.

---

## 🚀 Features

* Predicts **Rain / No Rain**
* Clean and beginner-friendly implementation
* Uses real-world weather data
* Easy to extend and deploy

---

## 🛠️ Technologies Used

* Python
* NumPy
* Pandas
* Scikit-learn
* Matplotlib (optional for visualization)

---

## 📊 Dataset

The dataset includes weather-related features such as:

* Temperature
* Humidity
* Wind Speed
* Pressure
* Outlook / Conditions

---

## ⚙️ How It Works

1. Data preprocessing (handling missing values, encoding)
2. Feature selection
3. Model training using ML algorithm
4. Model evaluation
5. Prediction output (Rain / No Rain)

---

## ▶️ How to Run the Project

1. Clone the repository:
   git clone https://github.com/AMINACHATTHA/rain_prediction_model_using_ml

2. Navigate to the project folder:
   cd rain-prediction-ml

3. Install dependencies:
   pip install -r requirements.txt

4. Run the model:
   python train.py
